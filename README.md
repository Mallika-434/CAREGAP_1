## Overview

CareGap is a clinical decision support dashboard built for nurse case managers managing patients with chronic conditions. It ingests synthetic EHR data from the Synthea generator (California population, ~20,000 patients) and provides real-time risk stratification, ML-powered disease progression prediction, and AI-assisted care recommendations.

---

## Features

### Population Dashboard
- Cohort-level statistics across chronic, at-risk, pediatric, and deceased patients
- HbA1c distribution and Systolic BP distribution charts
- Care gap cascade analysis (overdue labs, missing medications)

### Analytics Explorer
- Live filtering by cohort, gender, age, and condition
- Predictive Modeling tab supporting all three cohorts:
  - **Chronic** — 6-month progression risk with 3-model ensemble
  - **At-Risk** — HTN and T2D onset risk with 6-model ensemble
  - **Pediatric** — CDC age/gender-adjusted BMI assessment
- Model Impact Matrix with risk attribution (Sugar, BP, Other Factors)
- Layman Risk Summary generated by local LLM

### Patient Profiles
- Full clinical history, conditions, observations, encounters
- Risk Assessment with rule-based scoring (ACC/AHA 2023, ADA 2024)
- Trends, Urgent Care locator, History tabs
- AI Suggestions tab with quick clinical question chips

### Action Required (Triage)
- Emergency queue: SBP ≥ 160 or HbA1c ≥ 9.0
- Urgent queue: active chronic conditions with uncontrolled labs
- ML risk scores for adults, BMI category labels for pediatric patients
- Resource Forecast: beds, ICU rooms, nurse staffing for 30-day window

### AI Assistant
- Global chatbot (bottom-right) accessible from any page
- Sidebar AI Suggestions page for patient-specific consultations
- RAG pipeline: FAISS vector search → Ollama phi3 → Gemini Flash → rule-based fallback
- Explain This button on all prediction panels

## ML Models

### Chronic Disease Progression (3 models)
Trained on 6,267 chronic patients · 19 features · outcome-based clinical labels · SMOTE balancing · 5-fold CV

| Model | Accuracy | F1 |
|-------|----------|----|
| Lasso Logistic Regression | 71.1% | 53.5% |
| Random Forest (300 trees) | 77.6% | 62.0% |
| GradientBoosting | 86.1% | 62.0% |

### Disease Onset Models (6 models)
Trained on 16,821 at-risk patients · 13 features · compound clinical labels (HTN + T2D separately)

| Condition | Models |
|-----------|--------|
| Hypertension onset | Lasso LR, Random Forest, GradientBoosting |
| Type 2 Diabetes onset | Lasso LR, Random Forest, GradientBoosting |

### Pediatric BMI Assessment
CDC age/gender-adjusted thresholds across age groups 2-5, 6-11, 12-17.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Backend | Django 5.2 + Django REST Framework |
| Database | SQLite (seeded) + DuckDB (analytics) |
| ML | scikit-learn 1.8 (Lasso, Random Forest, GradientBoosting) |
| Vector Search | FAISS + sentence-transformers (all-MiniLM-L6-v2) |
| LLM | Ollama phi3 (local) + Gemini Flash (fallback) |
| Frontend | Vanilla JS + Chart.js |
| Static Files | WhiteNoise |
| Deployment | Hugging Face Spaces (Docker) |

## Local Setup

```bash
# Clone the repo
git clone https://github.com/Mallika-434/CAREGAP_1.git
cd CAREGAP_1

# Create virtual environment
python -m venv venv
venv\Scripts\activate  # Windows

# Install dependencies
pip install -r requirements.txt

# Create .env file
echo DB_PATH=db_demo.sqlite3 > .env
echo DEBUG=True >> .env
echo GEMINI_API_KEY=your_key_here >> .env

# Run migrations and seed data
python manage.py migrate
python manage.py seed_demo_data

# Build RAG index
python manage.py build_rag_index

# Warm cache and train models
python manage.py setup_demo

# Start server
python manage.py runserver
```

Open [http://127.0.0.1:8000](http://127.0.0.1:8000)
